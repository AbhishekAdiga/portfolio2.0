## ğŸ¨ Customization

### Colors

The color scheme can be customized in `tailwind.config.js`. The default uses a blue accent color with proper dark mode support.

### Content

Update the following files to customize content:
- `src/components/Hero.tsx` - Personal information and intro
- `src/components/About.tsx` - About section and skills
- `src/components/Projects.tsx` - Project showcase
- `src/components/Contact.tsx` - Contact information

### Images

Replace placeholder images in the `public` directory:
- Profile image for hero section
- Project screenshots
- Blog post images
